## 🚀 How to Use the Fixer

1. Download the latest **FixWave.bat** from this repository (Releases tab recommended).
2. **Right-click** `FixWave.bat` → **Run as administrator**.
3. Use the menu to fix your issues:

### Available Options

| Option | Description |
|--------|-------------|
| **1** Install Wave | Full fresh install + dependencies + Defender exclusions |
| **2** Fix Module Error | Deletes corrupted Wave folders + redownloads module |
| **3** Fix Loader Error / Initializing Issue | Fixes loader + WebView2 problems |
| **4** Fix Dependencies | Reinstalls all .NET, Visual C++, and runtimes |
| **5** Fix Failed to Generate HWID | Repairs WMI repository and classes |
| **6** Fix Invalid/Expired License | Time sync + DNS flush |
| **7** Install Cloudflare WARP | VPN/ISP bypass |
| **8** Whitelist in Windows Defender | Adds full exclusions for Wave |
| **9** Install Roblox Bootstrapper | WaveStrap, Bloxstrap, or Fishstrap |
